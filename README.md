## __libnnue__
A header only C++17 library for NNUE stuff.

---

## __What is NNUE?__
An Efficiency Updatable Neural Network (NNUE) can be used in a game engine as an evaluation function, among other things.

---

## __Requirements__
- A C++17 compiler

---

## __Tests__
